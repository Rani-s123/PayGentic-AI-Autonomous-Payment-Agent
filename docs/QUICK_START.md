# ⚡ Quick Start Guide: PayGentic

Follow these steps to get your AI Payment Agent up and running in under 5 minutes.

## 1. Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Locus API Credentials (get them from the Locus dashboard)

## 2. Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables:
   - Create a `.env` file.
   - Add your `LOCUS_API_KEY` and `LOCUS_SECRET_KEY`.
4. Start the server:
   ```bash
   npm start
   ```

## 3. Frontend Setup
1. Open a new terminal and navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 4. Test the Agent
1. Open `http://localhost:5173` in your browser.
2. Type a message in the chat:
   - *"Process $100 via UPI"*
   - *"Show my recent transactions"*
3. Watch the dashboard update in real-time!
